# Государственный оркестр народных инструментов Республики Дагестан — лендинг

Готовый одностраничный сайт в бело-синем официальном стиле.

Структура:
- `index.html` — основная страница;
- `assets/img/conductor.png` — фото дирижёра;
- `assets/img/orchestra.png` — фото оркестра.

Чтобы заменить изображения, сохраните новые файлы с теми же именами в папке `assets/img/`.
